# PHP运动刷步助手WEB版

#### 介绍
PHP运动刷步助手WEB版，基于小米运动接口协议刷步，步数可秒同步至微信、支付宝等平台

#### 更新日志


优化错误提示信息

优化提示弹窗为layer弹窗

优化源码目录

#### 软件架构
PHP

#### 安装教程

1.  上传源码解压即可使用

#### 使用说明

1.下载zepp Life。

2.从应用商店下载小米运动App，打开软件通过手机号注册登录。

3.登录之后，点击我的->第三方接入->绑定需要同步的第三方平台。

4.回到提交网页版，登陆，提交步数即可同步至你绑定的所有平台。
